# Topic History
各タスクで使用済みのトピックを記録し、重複を防ぐ。

**新規問題作成時のルール:**
1. トピック名が一致するものは絶対に使わない
2. キーワードが3つ以上重複するトピックは避ける（例: chocolate + trade + Europe が既出なら cocoa + commerce + European も NG）
3. 同じ学問分野が連続しないようにする（例: 生物学 → 歴史 → 物理学 のように分散）
